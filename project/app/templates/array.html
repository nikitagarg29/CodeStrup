Arrays

An array is a collection of items stored contiguously in the memory.

Index:

  1. Elements in arrays can be accesssed from their index values
  2. For exmple the first element is stored at 0th index, the second at 1, the third at 2 and so on....

Common Operations

Value assignment

In order to assign a value to a particular element in the array we need to access it by it's index value. So

cin >> arr[i];
}

Searching in an Array

Linear Search

The simplest way to search for a value in an array is to traverse through the array element by element until we find the value we are looking for.


void Linear_Search(int arr[], int size, int value) {
for (int i = 0; i < size(); i++)
if (arr[i] == value)
return i;
return -1; }

Binary Search

The linear search method can be time consuming hence a better method to search for an element in an array is binary search.

However, unlike the linear search method this only works in sorted arrays. As at each stage we are narrowing down the region in which the element might be found, therefore if we were to implement this in an unsorted array then we would not be able to narrow down the region where the element may be found.

In binary search we initialize the variables:

  1. low: as the lowest index value of the array
  2. high: as the highest index value of the array
  3. mid: as the middle most index ( (lo + high) / 2 ) value in the array

We then run a loop wherein we check if the element at mid is equal/greater/smaller compared to the value we are searching for. If-

  1. arr[mid] > value: we shift our high to mid - 1 as we now know that we the element is in the low to middle region
  2. arr[mid] < value: we shift our low to mid + 1 as we now know that we the element is in the middle to high region
  3. arr[mid] == value: we have found our element and return mid as the location


void Binary_Search(int arr[], int size, int value) {
int low = 0, high = size - 1;
int mid;
while (low <= high) {
mid = (high + low) / 2;
if (arr[mid] > value)
high = mid - 1;
else if (arr[mid] < value)
low = mid + 1;
else
return mid;
}
return -1;
}

Sorting an Array

Bubble Sort

Bubble sort is one of the easiest sorting methods.

It is based on the principle that we continue swapping adjacent elements until all the elements are in order.


void bubbleSort(int arr[], int n) {
int i, j;
for (i = 0; i < n-1; i++)
for (j = 0; j < n-i-1; j++)
if (arr[j] > arr[j+1])
swap(arr[j], arr[j+1]);
}

Selection Sort

In selection sort we traverse through the array looking for the minimum element and put it in the begining of the unsorted part of the array. Therefore by the doing this we sort the elements till there is no unsorted part left.

We then run a loop where we traverse through the array, we run a child loop that traverses thrugh the elements following the element in the parent loop.

We then find the minimum element in the unsorted subarray and swap it with the element in the begining.


void Selection_Sort(int arr[], int n) {
int i, j, min_idx;
for (i = 0; i < n-1; i++)
{
min_idx = i;
for (j = i+1; j < n; j++)
if (arr[j] < arr[min_idx])
min_idx = j;
swap(&arr[min_idx], &arr[i]);
}
}
}

Insertion Sort

In this method we divie the array into sorted and unsorted parts and place the element from the unsorted part in the correct position in the sorted part.

We then run a loop where we traverse through the array.

We then compare the value of the element with it's predecessers and if the elements are greater than the value then we move them one index ahead to make spae for the element to be put into the correct position.


void Insertion_Sort(int arr[], int n) {
int i, j, element;
for (i = 1; i < n-1; i++)
{ j = i - 1, element = arr[i];
while (j >= 0 && arr[j] > element) {
arr[j + 1] = arr[j];
j = j - 1;
}
arr[j + 1] = element;
}
}
}
Algorithm Time Complexity Space Complexity
Linear Search O(n) O(1)
Binary Search O(Log(n)) O(1)
Bubble Sort O(n^2) O(1)
Selection Sort O(n^2) O(1)
Insertion Sort O(n^2) O(1)
