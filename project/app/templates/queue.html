Queues

A Queue is a linear structure which follows a particular order in which the operations are performed. It follows the First In First Out (FIFO) order. The least recently entered element is the first one to be removed.

Common Operations

Insertion - Enqueue

An element is always enqueued at the rear end of the queue. In a linear array based implementation of a queue, this enqueue operation is a constant time operation.


void enqueue(int x) {
if (is_full()) {
cout << "Queue is full!" << endl;
return;
}
r += 1;
arr[r] = x;
return;
}

Deletion - Dequeue

An element is always dequeued from the front of the queue. In a linear array based implementation of a queue, this operation is an O(n) operation as all elements in the queue need to be shifted back by one index every time an element is dequeued.


int dequeue() {
if (is_empty()) {
cout << "Queue is empty!" << endl;
return -1;
}
int front = arr[0];
for (int i = 0; i < r; i++) {
arr[i] = arr[i + 1];
}
r -= 1;
return front;
}
