INSERTION IN TREE

A tree data structure is built recursively i.e. we begin with the root node and then build its left and right subtrees by calling the function again. The function calls proceed in the similar fashion until we insert a stopping value (can be -1).

A tree supports 4 kinds of traversal routes that are :

  • PREORDER
  • POSTORDER
  • INORDER
  • LEVELORDER
 1. Preorder Traversal - In this method of traversal the root of the tree is visited first then its left child and then the right child.
void preorder(node*root){
if(root==NULL){
return NULL;
}
cout<data;
preorder(root->left);
preorder(root->right);
}

2. Postorder traversal - In this method of traversal the left node is visted first then the right node and at last the root node.
void postorder(node*root){
if(root==NULL){
return NULL;
}
postorder(root->left);
postorder(root->right);
cout<data;
}

3. Inorder Traversal- In this method the left node is visited first then the root node and then the right node.
void inorder(node*root){
if(root==NULL){
return NULL;
}
inorder(root->left);
cout<data;
inorder(root->right);
}

4. Levelorder Traversal - In this method the nodes are are processed (visited) level by level from left to right.
